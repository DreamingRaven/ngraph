Distributed Training
####################

.. important:: Distributed training for CPU backend is not supported. Distributed 
   training support is provided only with the Intel® Nervana™ Neural Network Processor
   for Training (NNP-T).
   

Distributed training with Docker/Kubernetes is available with Intel® Nervana™ Neural 
Network Processor for Training (NNP-T).  See the system software and User Guide for 
details.
